# 🚀 TP React Débutant - Gestion d'état et Context API

[![React](https://img.shields.io/badge/React-18.2.0-61dafb)](https://reactjs.org/)
[![JavaScript](https://img.shields.io/badge/JavaScript-ES6-f7df1e)](https://developer.mozilla.org/fr/docs/Web/JavaScript)

## 📋 Description

Ce projet est un TP pratique pour apprendre les concepts fondamentaux de React :
- ✅ Formulaires contrôlés et non-contrôlés
- ✅ Lifting state up (partage d'état parent-enfant)
- ✅ Context API pour la gestion d'état globale
- ✅ Design moderne et responsive

## 🎯 Fonctionnalités

| Fonctionnalité | Description |
|----------------|-------------|
| 📝 Formulaire contrôlé | Gestion des champs avec useState |
| ✏️ Formulaire non-contrôlé | Utilisation des refs (useRef) |
| 🌡️ Convertisseur de température | Conversion Celsius ↔ Fahrenheit |
| 👤 Profil utilisateur | Connexion/Déconnexion avec Context API |

## 🛠️ Technologies utilisées

- **React 18** - Bibliothèque UI
- **Hooks** - useState, useContext, useRef
- **CSS3** - Animations, Flexbox, Grid
- **Context API** - Gestion d'état globale

## 📦 Installation

### 1. Cloner le projet
```
npx create-react-app tp-react-debutant

cd tp-react-debutant

2. Installer les dépendances

npm install
3. Lancer l'application

npm start

L'application sera accessible à : http://localhost:3000
```
